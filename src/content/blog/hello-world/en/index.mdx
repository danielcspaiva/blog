## Example Localized Post

This is an example of a blog post written in English. You can add your content here.

### Features of Localization

- Each post can have its own language version
- Posts are filtered by the current locale
- URLs still follow the [locale]/blog/[slug] pattern

You can have completely different content in each language version.
